## 安装运行
本demo采用[R2框架](https://github.com/dog-days/r2)。
```
git clone https://github.com/sn-demo/r2-real-time-frontend.git
npm install
npm run ac
npm start
```
请确保后端端口号和前端一致，在浏览器中输入http://localhost:8888/,回车访问即可。
